Über diesen Test

Der Text basiert auf die Digitalisierung von Google und der OCR von Google. Der Eingescannte Text ist eine übersetzung von 1776. Es gibt noch eine ältere übersetzung von 1684. Diese von 1776 scheint mir aber die besser verständlichere.
